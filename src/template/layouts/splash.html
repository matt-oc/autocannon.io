What is autocannon ?

Autocannon is an HTTP/1.1 benchmarking tool inspired by wrk. Autocannon supports all HTTP verbs, custom HTTP headers and payloads. Autocannon is extremely fast because it uses an HDR histogram to memorize the data.

Autocannon is easily scriptable and embeddable. The benchmark results can be natively exported as JSON. Autocannon can also be used as a library, to create complex benchmarks.
